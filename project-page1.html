Symphony of blocks

My first project on Unity, I tried to do something simple and at the same time interesting. The essence of the game is to push the pink block outside the box, only horizontal and vertical blocks can be moved at the same time. The game also implements object physics, the advantage of the game is the ability to create more levels with different solutions.
